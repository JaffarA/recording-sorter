## Objectives

a script to rename recordings

renames files with no prefix+number i.e.

```
m1-good-voice.mp3
m-bad-voice.mp3
```

to ```$PREFIX + $LEN_OF_PREFIX_IN_FOLDER+1 + DESCRIPTOR```

```
m1-good-voice.mp3
m2-bad-voice.mp3
```

## Installation
